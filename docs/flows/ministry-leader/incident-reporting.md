# Ministry Leader Incident Reporting Flow

## Overview

Ministry leaders can create and submit incident reports for events that occur during their ministry activities.

## Flow Steps

1. **Access Incident Reporting**
   - Navigate to `/dashboard/incidents`
   - Click "Report Incident"

2. **Create Incident Report**
   - Enter incident details
   - Select affected children
   - Add description
   - Set severity level

3. **Submit Incident**
   - Review incident details
   - Submit report
   - Incident sent to admins for review
