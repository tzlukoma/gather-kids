# Ministry Leader Rosters Flow

## Overview

Ministry leaders can view ministry-specific rosters, filter by assigned ministries, check children in/out, and filter by grade and status.

## Flow Steps

1. **Access Rosters**
   - Navigate to `/dashboard/rosters`
   - View children in assigned ministries
   - Filtered by `assignedMinistryIds`

2. **Select Event**
   - Choose event (Sunday School, Children's Church, Teen Church)
   - Event selection filters available children

3. **Filter Children**
   - Filter by grade (optional)
   - Filter by status (All, Checked In, Checked Out)
   - Apply filters

4. **Check-in/Check-out**
   - Check children in
   - Verify PIN for check-out
   - View attendance status
