# Ministry Leader Registrations Flow

## Overview

Ministry leaders can view filtered registrations for their assigned ministries, review household information, and see registration details.

## Flow Steps

1. **Access Registrations**
   - Navigate to `/dashboard/registrations`
   - View registrations filtered by assigned ministries
   - Filtered by `assignedMinistryIds`

2. **Filter Registrations**
   - Filter by ministry
   - Filter by cycle
   - Filter by status

3. **View Registration Details**
   - Click on registration
   - View household information
   - See guardian details
   - Review child information
   - Check ministry enrollments
