# Ministry Leader Bible Bee Tracking Flow

## Overview

Ministry leaders can track Bible Bee progress for students in their assigned ministries, view scripture assignments, and monitor completion.

## Flow Steps

1. **Access Bible Bee Tracking**
   - Navigate to `/dashboard/bible-bee`
   - View students in assigned ministries
   - Filtered by ministry with 'bible-bee' code

2. **View Student Progress**
   - Select student
   - View assigned scriptures
   - See completion status
   - Review essay submissions

3. **Track Completion**
   - Monitor scripture completion
   - Review essay submissions
   - Track progress
