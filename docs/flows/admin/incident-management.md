# Admin Incident Management Flow

## Overview

Admins can review and resolve incidents, assign responses, and track incident status.

## Flow Steps

1. **Access Incident Management**
   - Navigate to `/dashboard/incidents`
   - View reported incidents
   - Filter by status, severity

2. **Review Incident**
   - Click on incident
   - View incident details
   - Review reporter information
   - Check related children

3. **Assign Response**
   - Assign incident to responder
   - Add notes
   - Update status

4. **Resolve Incident**
   - Mark incident as resolved
   - Add resolution notes
   - Close incident
