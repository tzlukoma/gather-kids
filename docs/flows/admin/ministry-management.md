# Admin Ministry Management Flow

## Overview

Admins can create and configure ministries, assign leaders, configure ministry groups, and set ministry codes and settings.

## Flow Steps

1. **Access Ministry Management**
   - Navigate to `/dashboard/ministries`
   - View list of all ministries

2. **Create Ministry**
   - Click "Create Ministry"
   - Enter ministry details (name, code, description)
   - Configure settings
   - Save ministry

3. **Edit Ministry**
   - Select ministry
   - Update details
   - Modify settings
   - Save changes

4. **Assign Leaders**
   - Select ministry
   - Add leader assignments
   - Link email addresses to ministry groups
   - Save assignments

5. **Configure Ministry Groups**
   - Create ministry groups
   - Add email addresses to groups
   - Link groups to ministries
   - Configure group settings
