# Admin Branding Flow

## Overview

Admins can customize application branding including logos, app name, color schemes, and UI elements.

## Flow Steps

1. **Access Branding**
   - Navigate to `/dashboard/branding`
   - View current branding settings

2. **Upload Logo**
   - Upload logo image
   - Set logo URL
   - Configure logo-only mode

3. **Configure App Name**
   - Set application name
   - Update display name

4. **Customize Colors**
   - Set primary colors
   - Configure theme colors
   - Preview changes

5. **Save Changes**
   - Preview branding changes
   - Save configuration
   - Changes apply system-wide
