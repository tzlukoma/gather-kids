# Admin Check-in Management Flow

## Overview

Admins have system-wide oversight of check-ins, can manage events, monitor attendance, and handle check-out verification.

## Flow Steps

1. **Access Check-in Management**
   - Navigate to `/dashboard/check-in`
   - View all check-ins across all events
   - See system-wide statistics

2. **Manage Events**
   - Create/edit events
   - Configure event settings
   - Set event timeslots

3. **Monitor Attendance**
   - View real-time attendance
   - See checked-in children
   - Monitor check-out status

4. **Handle Check-out Verification**
   - Assist with check-out issues
   - Override PIN verification if needed
   - Handle special cases
