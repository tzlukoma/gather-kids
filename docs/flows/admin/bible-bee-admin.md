# Admin Bible Bee Administration Flow

## Overview

Admins can set up Bible Bee years, import scriptures, configure divisions, track student progress, and review essay submissions.

## Flow Steps

1. **Access Bible Bee Administration**
   - Navigate to `/dashboard/bible-bee`
   - View Bible Bee cycles and years

2. **Create Bible Bee Year**
   - Create new competition year
   - Set year name and dates
   - Configure active status

3. **Import Scriptures**
   - Import scripture list for year
   - Assign scriptures to divisions
   - Configure scripture assignments

4. **Configure Divisions**
   - Create divisions (Primary, Junior, Senior)
   - Set grade ranges for divisions
   - Configure division settings

5. **Track Student Progress**
   - View enrolled students
   - Monitor scripture completion
   - Track essay submissions
   - Generate reports
