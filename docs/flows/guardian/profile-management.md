# Guardian Profile Management Flow

## Overview

Guardians can update household information, guardian details, child information, and emergency contacts.

## Flow Steps

1. **Access Profile Management**
   - Navigate from household portal
   - Select "Edit Profile" or "Update Information"

2. **Update Household Information**
   - Edit address
   - Update household name
   - Modify contact information

3. **Update Guardian Details**
   - Edit guardian information
   - Update phone numbers
   - Modify email addresses

4. **Update Child Information**
   - Edit child details
   - Update medical information
   - Modify allergies and special needs

5. **Update Emergency Contacts**
   - Edit emergency contact information
   - Update phone numbers
   - Modify relationships

6. **Save Changes**
   - Validate changes
   - Save updates
   - Confirm success
