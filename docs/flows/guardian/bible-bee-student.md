# Guardian Bible Bee Student Flow

## Overview

Guardians can track Bible Bee progress for their children, view assigned scriptures, mark scriptures as complete, and submit essays.

## Flow Steps

1. **Access Bible Bee Student**
   - Navigate to `/household/bible-bee`
   - View enrolled children
   - Select child

2. **View Assigned Scriptures**
   - See list of assigned scriptures
   - View scripture text
   - Check completion status

3. **Mark Scriptures Complete**
   - Mark scripture as complete
   - Update progress
   - Track completion

4. **Submit Essays**
   - Write essay for division
   - Submit essay
   - Track submission status

5. **View Progress**
   - See overall progress
   - View completion percentage
   - Track essay submissions
