Bible Bee Minimum Required Field - Mobile Input Fix

This demo shows the before and after behavior of the "Minimum Required" input field in the Bible Bee Divisions section.

❌ Before (Problem)

Uses type="number" - can show leading zeros on mobile that are hard to remove
Value: 0

✅ After (Fixed)

Uses type="text" with inputMode="numeric" - automatically removes leading zeros
Value: 0

Test Cases

Try these inputs in both fields to see the difference:

  • 05 - Should become 5 (fixed version removes leading zero)
  • 012 - Should become 12 (fixed version removes leading zeros)
  • 0 - Should stay 0 (both work correctly)
  • abc123 - Should become 123 (fixed version filters non-numeric)
  • 000 - Should become 0 (fixed version handles all zeros)
